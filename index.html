moodify.site

Moodify is built using Spotify's web API to dynamically generate a unique playlist for each user. Playlist generation starts with the user inputting their current mood. Once mood is selected, Moodify accesses the user's liked songs & creates a playlist of user's liked songs that are tagged with tags similiar to inputted mood.

Storm Bot

Storm Bot is a lightweight discord bot written in Python that uses Nextcord. Storm Bot revolves around async/await functions to handle user commands. Current features include a "Guess The Artist" game, logging anytime a message is deleted/edited, coinflipping, and more. I'm currently working on a music player feature for the bot! Storm Bot is named after my wonderful cat, Storm!
